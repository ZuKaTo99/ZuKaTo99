## Projects

### PMD - Project Manager Dashboard

Project Manager Dashboard is a MAUI Blazor Hybrid app for managing projects, project states and project folder checks.

**Tech:** C#, .NET MAUI, Blazor Hybrid, HTML, CSS, Windows Desktop, CommunityToolkit.Maui

* Project overview
* Project folder scanning
* File count and size analysis
* Clean German UI texts
* Future plans: project states, change reports, SQLite and Rust core integration

---

### TabellenSoftware / ExcelSoftware

A private C# / .NET Excel comparison tool with write-back logic, conflict handling and PDF report generation.

**Tech:** C#, .NET, WinForms, ClosedXML, QuestPDF, Excel processing, PDF reports

* Excel comparison
* Conflict detection
* Manual decisions
* Write-back preview
* Safe write-back into copied files
* PDF reports for unclear cases

---

### JUMP-and-RUN

A small C# WinForms jump-and-run mini-game with events, upgrades, high scores and progression systems.

**Tech:** C#, WinForms, .NET Framework, Game Logic

* Player movement
* Power-ups
* Events
* Achievements
* High scores
* Upgrade systems

---

### ScreenLocker

A small C# desktop tool that locks the screen as soon as the mouse is moved or a button is pressed.

**Tech:** C#, Windows Desktop, Input Detection

---

### RuneRunners UI / RunHUD

Unity UI work for a game frontend, including HUD elements and safe-area handling.

**Tech:** Unity, C#, UI Toolkit, UXML, USS

* HP display
* Score display
* Timer display
* Safe-area support
* Clean UI controller structure
