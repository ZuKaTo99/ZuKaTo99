## Projects

### PMD - Project Manager Dashboard
A desktop-oriented project management and analysis application built with .NET MAUI Blazor Hybrid.  
The goal of PMD is to provide a clean project overview, folder analysis, project history and future change tracking in a modern UI.

**Tech:** C#, .NET MAUI, Blazor Hybrid, HTML, CSS, Windows Desktop, CommunityToolkit.Maui

**Main features:**
- Project overview
- Folder scanning and analysis
- File count and size evaluation
- Clean German UI texts
- Planned expansion with project history, change reports, SQLite and Rust core integration

---

### TabellenSoftware / ExcelSoftware
A private C# / .NET desktop tool for structured Excel comparison, conflict handling and safe write-back workflows.  
The software is designed to support controlled data merging, manual review processes and PDF-based reporting.

**Tech:** C#, .NET, WinForms, ClosedXML, QuestPDF, Excel processing, PDF reporting

**Main features:**
- Excel comparison workflows
- Conflict detection and review
- Manual decision support
- Write-back preview
- Safe write-back into copied files
- PDF reports for unclear or complex cases

---

### JUMP-and-RUN
A small C# WinForms game project focused on gameplay systems, events, upgrades and progression.  
The project combines classic jump-and-run mechanics with score systems, achievements and player improvements.

**Tech:** C#, WinForms, .NET Framework, Game Logic

**Main features:**
- Player movement and game flow
- Power-ups and event systems
- Achievements and progression
- High scores and upgrade mechanics

---

### RuneRunners UI / RunHUD
A Unity UI project focused on building HUD elements and structured frontend logic for an in-game run screen.  
This includes safe-area support, clean controller structure and UI Toolkit-based implementation.

**Tech:** Unity, C#, UI Toolkit, UXML, USS

**Main features:**
- HP display
- Score display
- Timer display
- Safe-area support
- Structured UI controller setup

---

### ScreenLocker
A lightweight desktop utility that locks the screen immediately when mouse movement or input is detected.  
It was built as a small practical tool with a simple purpose and direct behavior.

**Tech:** C#, Windows Desktop, Input Detection
